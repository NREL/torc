# Tutorials

This section contains learning-oriented lessons to help you get started with Torc. Each tutorial walks through a complete example from start to finish.

**Tutorials:**

1. **Many Independent Jobs** - Create a workflow with 100 parallel jobs
2. **Diamond Workflow** - Fan-out and fan-in with file dependencies
3. **User Data Dependencies** - Pass JSON data between jobs
4. **Simple Job Parameterization** - Single parameter dimension sweep
5. **Advanced Parameterization** - Multi-dimensional hyperparameter grid search

Start with Tutorial 1 if you're new to Torc, then progress through the others to learn more advanced features.
